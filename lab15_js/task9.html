Створення Map з двох масивів (Перевірте консоль)

Функція `mapBuilder` об'єднує ключі та значення з двох масивів в об'єкт `Map`.

let keys = [1, 2, 3, 4];
let values = ["div", "span", "b", "i"];

let map = mapBuilder(keys, values);

console.log(map.size); // Очікується: 4
console.log(map.get(2)); // Очікується: "span"

Результати перевірки виведені в консоль браузера.
