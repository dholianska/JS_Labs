Результат Багаторівневої Деструктуризації Об'єкта (Перевірте консоль)

Вихідний код для деструктуризації:

let data = {
    names: ["Sam", "Tom", "Ray", "Bob"],
    ages: [20, 24, 22, 26],
};

let {
    names: [, name2, , name4],
    ages: [, age2, , age4]
} = data;

У консолі браузера ви побачите вивід: "Tom", 24, "Bob", 26.
