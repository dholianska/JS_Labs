Функція `mul()` для добутку чисел (Перевірте консоль)

Функція використовує оператор залишку `...args` для збору всіх аргументів у масив.

function mul(...args) {
    // ... ваш код для фільтрації та множення
}

console.log(mul(1, "str", 2, 3, true)); // Очікується: 6
console.log(mul(null, "str", false, true)); // Очікується: 0

Результати викликів функції виведені в консоль браузера.
